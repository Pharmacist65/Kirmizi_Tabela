# Kırmızı Tabela Game Assets

This directory contains project-owned runtime assets for the playable WebGL prototype.

- `models/*.glb` files are generated by `npm run assets:generate`.
- `asset-manifest.json` is generated by the same script and maps logical model names to public URLs.
- The current models are first-pass authored placeholders created from local Three.js geometry. They are safe to keep in the public repository and can be replaced later with Blender-authored, licensed, or commissioned GLB assets without changing the game loop.

No Abeto/Messenger source files, models, textures, or bundled assets are copied into this project.
